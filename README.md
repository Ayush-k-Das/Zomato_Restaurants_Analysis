# Zomato Restaurants Analysis — Power BI + ML

This repository contains a complete data analysis and machine learning pipeline for the Zomato restaurants dataset, plus a Power BI dashboard template for visualization.

## Contents
- `notebooks/` — exploratory notebooks (cleaning, EDA, feature engineering, modeling)
- `src/` — reusable Python modules
- `data/` — place your `zomato.csv` here (NOT tracked)
- `models/` — saved model artifacts
- `powerbi/` — Power BI template and instructions
- `reports/` — final writeups and findings

## Objectives
1. Build an interactive Power BI dashboard for Zomato EDA and KPIs.
2. Predict restaurant `Rating` from features (cuisines, cost, votes, location, delivery options) using machine learning (Random Forest chosen as final model).
